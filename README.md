# AI Contour - Автоматическое оконтурирование органов риска на КТ (MVP)

Этот проект представляет собой приложение для автоматической сегментации органов риска (OAR) на КТ-исследованиях и последующего экспорта полученных контуров в стандартный DICOM RTSTRUCT файл, готовый к импорту в системы планирования лучевой терапией (TPS).

Приложение поддерживает два режима запуска: **графический интерфейс (GUI)** для удобной работы врача и **командную строку (CLI)** для автоматизации.

---

## 🚀 Основные возможности

1. **Двойной режим запуска:**
   * **Графический интерфейс (GUI):** Запускается по умолчанию при вызове без параметров. Выполнен в элегантном современном темном стиле (Dark Mode).
   * **Командная строка (CLI):** Используется при передаче аргументов (например, `-i` и `-o`) для удобной автоматизации на серверах или скриптах.
2. **Изолированный запуск ИИ (100% стабильность):**
   * Вычисления `TotalSegmentator` вынесены во внешний изолированный системный процесс с неблокирующим чтением логов в реальном времени. Это гарантирует отсутствие падений из-за конфликтов Intel OpenMP/PyTorch на Windows.
   * GUI запускается мгновенно (менее 0.5 сек) и потребляет минимум ресурсов в режиме ожидания.
   * Вся оперативная память, занятая нейросетью, гарантированно возвращается операционной системе сразу после окончания расчетов.
3. **Умное слияние контуров (Merge Mode):**
   * Приложение сканирует папку КТ на наличие существующего RTSTRUCT-файла разметки врача.
   * В режиме слияния ИИ добавляет новые органы, не затирая ручные контуры врача.
   * Если орган уже размечен врачом (например, мочевой пузырь), ИИ-контур добавляется с суффиксом `(AI)` (например, `Urinary Bladder (AI)`), сохраняя оригинальную разметку врача в безопасности.
4. **Система пресетов OAR и анатомическая группировка:** 
   * **Анатомические группы:** Все органы в интерфейсе структурированы по областям («Голова и шея», «Грудная клетка», «Брюшная полость», «Малый таз») с красивыми невыбираемыми заголовками в стиле Dark Mode.
   * **Быстрый выбор:** Поддержка готовых пресетов органов риска (включая новый пресет для головы и шеи).
   * **Умная синхронизация:** Органы, входящие в несколько областей одновременно (например, трахея и пищевод), автоматически синхронизируют состояние своих чекбоксов в реальном времени, обеспечивая идеальный UX.
5. **Конвертация DICOM ➔ NIfTI:** Сборка КТ-срезов в единый 3D-объем с сохранением пространственных метаданных (`dicom2nifti`).
6. **Сборка масок в DICOM RTSTRUCT:** Корректное транспонирование воксельных масок и запись в стандартный файл RTSTRUCT (`rt-utils`) с жесткой привязкой к координатам оригинального КТ.

## 🛠️ Установка и быстрый старт

### 1. Создание и активация виртуального окружения
Откройте терминал (например, Windows PowerShell) в папке проекта:
```powershell
# Создание venv
python -m venv venv

# Активация venv
.\venv\Scripts\Activate.ps1
```

### 2. Установка зависимостей
Обновите пакетный менеджер и установите все необходимые библиотеки:
```powershell
python -m pip install --upgrade pip
pip install -r requirements.txt
```

### 3. Запуск приложения

#### Режим графического интерфейса (GUI)
Просто запустите скрипт без дополнительных аргументов:
```powershell
python auto_contour.py
```
Откроется современное темное GUI-окно, в котором вы сможете указать папки, выбрать органы и запустить оконтурирование в один клик.

#### Режим командной строки (CLI)
Для запуска в фоновом/тихом режиме передайте пути к снимкам и выходной папке:
```powershell
python auto_contour.py -i "C:\path\to\ct_dicom" -o "C:\path\to\output_dir" -p abdominal_oar
```

**Доступные пресеты в CLI (`-p` / `--preset`):**
* `head_neck_oar` — Органы головы и шеи (спинной мозг, щитовидная железа, череп, трахея, пищевод).
* `abdominal_oar` — Органы брюшной полости (селезенка, почки, печень, желудок, аорта, мочевой пузырь, сердце).
* `thoracic_oar` — Органы грудной клетки (сердце, легкие, трахея, аорта, пищевод).
* `pelvis_oar` — Органы малого таза (мочевой пузырь, простата, прямая кишка, кишечник, бедренные и тазовые кости, крестец).
* `all` — Экспортировать все найденные ИИ-моделью структуры.
